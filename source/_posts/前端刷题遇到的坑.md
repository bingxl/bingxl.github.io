## js部分
```
1. 
var bb = 1;
function aa(bb) {
    bb = 2;
    alert(bb);
};
aa(bb);
alert(bb);

2.
var bb = 1;
function aa() {
    bb = 2;
    alert(bb);
};
aa(bb);
alert(bb);
```
刚开始1处的代码我认为弹出的是 2,2 ；被我做急了，只看到函数内部的bb没有使用var定义，忽略了参数里面已经有了bb，最后函数内部的bb就是函数内部的局部变量了，所以结果因该是 2,1

2处的代码执行后就是2,2 因为函数参数里面没有了bb变量，函数内部对bb赋值改变了全局变量bb的值
